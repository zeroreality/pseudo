Pseudo test case: forms

document.forms[0].state()
Saves the disabled states of the form elements for later when re-enabling
document.forms[0].disable()
disables the form and saves state
document.forms[0].disable(true)
disables the form; does not save state
document.forms[0].enable()
enables the form from saved state; if no state was set then enables form element
document.forms[0].enable(true)
enables the form from regardless of state
console.log(document.forms[0].data())
provides an object representing the form element values
console.log(document.forms[0].data("joined"))
provides an object representing the form element values; all values are strings, all strings are comma separated for names with multiple values
console.log(document.forms[0].data("query"))
provides a query string ready for submitting via POST or GET
